# Linux Portfolio

This repository documents my structured Linux learning journey,
focused on system administration, troubleshooting, and automation.

## Environment
- OS: Ubuntu Server 22.04 LTS
- Shell: Bash

## Scope
- Linux filesystem and permissions
- Process and resource management
- Package management
- Networking fundamentals
- Storage and LVM
- Systemd and Services
- Logging and troubleshooting
- Bash scripting
- Basic security hardening

## Structure
- notes/ — theory and concepts
- labs/ — hands-on practical labs
- scripts/ — automation scripts
- incident-reports/ — troubleshooting case studies

## Goal
Build a strong Linux foundation for infrastructure, cloud, and DevOps roles.
