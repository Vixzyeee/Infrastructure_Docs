# Python Learning Journey

This repository contains my structured learning path for Python, starting from foundational concepts and progressing toward automation, scripting, and backend fundamentals. 
It includes notes, exercises, and small projects designed to develop strong problem-solving skills and practical programming experience.

---

## Objectives

- Build solid understanding of Python basics  
- Develop consistent problem-solving habits through exercises  
- Apply concepts in small, focused projects  
- Learn practical automation techniques (HTTP requests, JSON handling, SSH workflows)  
- Prepare for backend and infrastructure-related development

---

## Requirements

- Python **3.10+** recommended  
- Optional tools:
  - VS Code / PyCharm / any preferred IDE
  - `pip` for installing packages
- Additional dependencies listed in **requirements.txt**

---

## Repository Structure

notes/ Concept explanations and theory notes
exercises/ Practice problems grouped by topic
projects/ Mini-projects applying real use cases
automation/ Scripts for HTTP, SSH, JSON, file handling
sandbox/ Experimental scratchpad for testing ideas
