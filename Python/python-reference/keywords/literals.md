# Literal Keywords

## What is this?
Literal keywords represent constant values that are built into the Python language.

---

## True
**Purpose:** Represent the boolean value true.

```python
print(True)
```

**Output:**

```text
True
```

**Notes:**

- Represents a truth value
- Used in boolean expressions and conditions

---

## False
**Purpose:** Represent the boolean value false.

```python
print(False)
```

**Output:**

```text
False
```

**Notes:**

- Represents a false value
- Used in boolean expressions and conditions

---

## None
**Purpose:** Represent the absence of a value.

```python
x = None
print(x)
```

**Output:**

```text
None
```

**Notes:**

- Represents a null or missing value
- Commonly used as a default or placeholder
