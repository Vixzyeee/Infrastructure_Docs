# Class Keywords

## What is this?
The class keyword is used to define a new class in Python.

---

## class
**Purpose:** Define a class.

```python
class Person:
    def __init__(self, name):
        self.name = name

p = Person("Alice")
print(p.name)
```

**Output:**

```text
Alice
```

**Notes:**

- Used to create a new class definition
- Class body is defined by indentation
- Instances are created by calling the class
