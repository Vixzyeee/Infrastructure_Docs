# Python Keywords Reference

This directory contains a categorized reference of Python language keywords.
Each file groups related keywords by their functional domain.

The purpose of this directory is to provide a **clean, keyword-focused reference**,
not tutorials or conceptual explanations.

---

## Structure Overview

- `conditionals.md`  
  Conditional decision-making keywords (`if`, `elif`, `else`)

- `loops.md`  
  Looping and iteration keywords (`for`, `while`, `break`, `continue`, `pass`)

- `functions.md`  
  Function-related keywords (`def`, `return`, `lambda`, `yield`)

- `classes.md`  
  Class definition keyword (`class`)

- `exceptions.md`  
  Exception handling keywords (`try`, `except`, `finally`, `raise`, `assert`)

- `imports.md`  
  Import system keywords (`import`, `from`, `as`)

- `logical.md`  
  Logical and membership keywords (`and`, `or`, `not`, `is`, `in`)

- `async.md`  
  Asynchronous execution keywords (`async`, `await`)

- `context.md`  
  Context management keyword (`with`)

- `scope.md`  
  Scope and namespace control keywords (`global`, `nonlocal`, `del`)

- `literals.md`  
  Literal keywords (`True`, `False`, `None`)

---

## Scope Rules

- Only **Python language keywords** are documented here
- Built-in functions, methods, and operators are documented elsewhere
- No patterns, best practices, or deep theory are included

This directory is intended to be a **precise reference**, not a learning tutorial.
