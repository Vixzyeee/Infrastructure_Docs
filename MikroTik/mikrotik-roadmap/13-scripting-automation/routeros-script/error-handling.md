## What this covers

Error handling prevents silent failure.<br>
Scripts must fail safely.

## Key concepts

- :do / :on-error
- Logging errors
- Safe exit

## Why it matters

Silent errors hide problems.<br>
Visibility is mandatory.

## Common mistakes

- Ignoring return values
- No error logs

## Real-world example

Backup script fails silently.<br>
No usable backup exists.
