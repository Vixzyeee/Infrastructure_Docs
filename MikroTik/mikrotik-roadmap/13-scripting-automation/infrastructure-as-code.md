## What this covers

Infrastructure as Code treats config as source.<br>
Versioned and repeatable.

## Key concepts

- Declarative mindset
- Git as source of truth
- Drift detection

## Why it matters

Manual changes break automation.<br>
IaC enforces discipline.

## Common mistakes

- Mixing manual and automated changes
- No review process

## Real-world example

Automation applies config.<br>
Manual change overrides it later.
