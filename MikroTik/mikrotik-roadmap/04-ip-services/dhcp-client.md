## What this covers

DHCP Client automates WAN configuration.<br>
It injects IP, route, and DNS.

## Key concepts

- Default route injection
- DNS and NTP options
- Interface binding

## Why it matters in MikroTik

Multiple DHCP clients can break routing.<br>
Order and distance matter.

## Common mistakes

- Forgetting DHCP adds default route
- Running DHCP client on wrong interface

## Real-world example

Traffic exits wrong ISP.<br>
Unexpected default route added.
