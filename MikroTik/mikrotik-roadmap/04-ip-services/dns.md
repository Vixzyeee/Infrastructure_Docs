## What this covers

DNS resolves names to IP addresses.<br>
Without it, users think internet is down.

## Key concepts

- Recursive vs forwarder
- Cache behavior
- Static DNS entries

## Why it matters in MikroTik

Router-level DNS reduces client latency.<br>
Also centralizes control.

## Common mistakes

- Using ISP DNS blindly
- Forgetting to allow DNS in firewall

## Real-world example

Ping by IP works.<br>
Website names fail to resolve.
