## What this covers

DHCP Relay forwards requests to remote servers.<br>
Used in segmented networks.

## Key concepts

- Relay agent
- Hop count
- Server reachability

## Why it matters in MikroTik

Large networks centralize DHCP.<br>
Relay enables that design.

## Common mistakes

- Forgetting relay interface
- Blocking DHCP packets with firewall

## Real-world example

Clients stuck at “obtaining IP”.<br>
Relay traffic blocked.
