## What this covers

NTP synchronizes system time.<br>
Logs depend on accurate time.

## Key concepts

- Client vs server mode
- Timezone setting
- Drift correction

## Why it matters in MikroTik

Wrong time breaks logs and certificates.<br>
Debugging becomes unreliable.

## Common mistakes

- Ignoring timezone
- Relying on manual time

## Real-world example

Log order incorrect.<br>
Clock never synced.
