## What this covers

NAT modifies packet addresses.<br>
It does not secure traffic.

## Key concepts

- src-nat vs dst-nat
- Masquerade behavior
- NAT order

## Why it matters in MikroTik

Misunderstanding NAT causes broken apps.<br>
Especially port forwarding.

## Common mistakes

- Using NAT as firewall
- Overusing masquerade

## Real-world example

Port forward works internally only.<br>
Hairpin NAT missing.
