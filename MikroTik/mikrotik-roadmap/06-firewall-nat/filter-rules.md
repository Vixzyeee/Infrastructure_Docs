## What this covers

Filter rules decide allow or deny.<br>
They enforce security policy.

## Key concepts

- Accept vs drop
- Rule order
- Logging impact

## Why it matters in MikroTik

Bad filters lock you out.<br>
Good filters are invisible.

## Common mistakes

- Drop-all rule placed too early
- Over-logging everything

## Real-world example

Router inaccessible remotely.<br>
INPUT drop rule misordered.
