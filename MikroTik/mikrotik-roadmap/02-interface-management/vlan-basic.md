## What this covers

VLAN separates traffic logically.<br>
Same cable, different networks.

## Key concepts

- Access vs trunk
- VLAN ID
- Tagged vs untagged

## Why it matters in MikroTik

VLAN design affects routing and firewall.<br>
Bad VLAN = bad security.

## Common mistakes

- Mixing access and trunk logic
- Forgetting VLAN tagging on uplinks

## Real-world example

Client has link but no IP.<br>
VLAN tag missing on trunk.
