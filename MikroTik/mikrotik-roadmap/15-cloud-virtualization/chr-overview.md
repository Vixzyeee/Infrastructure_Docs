## What this covers

Cloud Hosted Router (CHR) is RouterOS for virtual platforms.<br>
No switch chip. No ASIC.

## Key concepts

- License levels
- Virtual NIC performance
- CPU-bound forwarding

## Why it matters

CHR behaves differently from hardware.<br>
Expectations must adjust.

## Common mistakes

- Expecting hardware offload
- Undersizing CPU

## Real-world example

Throughput low.<br>
CPU hits 100 percent.
