## What this covers

DHCP Client is commonly used for WAN.<br>
It controls IP, gateway, and DNS.

## Key concepts

- Default route injection
- DNS and NTP options
- Interface binding

## Common mistakes

- Forgetting DHCP Client adds default route
- Multiple WANs with uncontrolled routes

## Real-world example

Traffic exits wrong WAN.<br>
Unexpected default route added.
