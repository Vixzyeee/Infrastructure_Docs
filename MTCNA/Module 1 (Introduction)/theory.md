# What is MikroTik

## Company Profile:
- Based in Riga, Latvia (Capital of Latvia)
- European company with 100% European manufactured products
- Location borders: Estonia, Russia, Belarus, and Lithuania
  
## Business Focus:
- Router software development
- Hardware manufacturing
- Creates products for ISPs, companies (all sizes), and individuals
  
## Company Mission:
- Make internet technologies faster, powerful, and affordable
- Provide cost-effective solutions compared to Cisco, Juniper
- Offer comprehensive functionality at affordable prices

  
# Mikrotik History Timeline
- 1995: Started as wireless ISP providing internet services via wireless
- 1997: Created RouterOS software for network management (x86 architecture)
- 2002: Launched first RouterBOARD hardware
- 2006: Initiated MUM (MikroTik User Meeting) gatherings
    - Team visits countries for product demonstrations
    - New features and hardware announcements
    - Consultant and trainer presentations
    - Suspended after COVID-19 pandemic


# MikroTik RouterOS (Operating System)

## Technical Foundation:
- Based on Linux kernel
- Operating system for MikroTik routers
- Can be installed on PC hardware (old computers)

## Core Capabilities:
- Routing Functions: Static, OSPF, RIP, BGP, MPLS protocols
- Network Services: Bandwidth shaping, Quality of Service
- Security Features: Transparent packet filtering, firewall
- Wireless Support: 802.11 a/b/g/n/ac/ax standards
- VPN Technologies: PPTP, L2TP, WireGuard, Zerotier
- Advanced Features: Hotspot, user management, PPPoE, VPLS, traffic engineering

## Licensing Model:
- One-time purchase with router
- No annual licensing fees
- Lifetime validity
- Free updates to latest versions

# MikroTik Product Types

## Board-Only Solutions
- Circuit board with basic components
- CPU, RAM, interfaces (Gigabit Ethernet, SFP cage)
- Wi-Fi capabilities (5GHz ac)
- RouterOS license included
- Requires custom case and power adapter
- Build-your-own router approach

## Integrated Solutions
- Complete RouterBOARD products
- Includes cases and power adapters
- Pre-configured with basic settings
- Plug-and-play functionality
- Ready-to-use out of box
- Popular choice for most users
