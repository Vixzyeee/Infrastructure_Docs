# top and htop

## Purpose
Monitor processes in real time.

## Core Concept
`top` displays a live view of system processes,
including CPU and memory usage.

`htop` is an enhanced alternative
with a more readable interface
and interactive controls.

## Engineer Notes
Use `top` when htop is unavailable.
Use `htop` when you value your eyesight.

Real-time monitoring is essential
for diagnosing performance issues.

## Summary
- Real-time process monitoring
- Shows CPU and memory usage
- htop is more user-friendly
