# Package Repositories

## Purpose
Explain where packages come from.

## Core Concept
Repositories are servers that host software packages.
Package managers download packages from configured repositories.

Repository configuration determines:
- available software
- package versions
- security updates

## Engineer Notes

## Summary
- Repositories provide packages
- Configuration affects system stability
- Fewer repositories is usually better
