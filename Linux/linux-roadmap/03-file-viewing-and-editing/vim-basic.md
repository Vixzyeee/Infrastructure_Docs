# Vim Basics

## Purpose
Introduce basic usage of the Vim editor.

## Core Concept
Vim is a modal text editor.
It separates navigation, insertion, and command execution.

Basic modes:
- Normal mode
- Insert mode
- Command mode

## Engineer Notes
You do not need to master Vim immediately.
Knowing how to open, edit, save, and exit is enough at first.

## Summary
- Modal text editor
- Extremely powerful once learned
